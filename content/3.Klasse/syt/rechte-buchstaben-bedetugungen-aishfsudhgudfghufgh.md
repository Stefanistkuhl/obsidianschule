```sh
d|r-x|r-x|r-x
```
die striche sind zum abtrennen der teile da

1.teil → user permissions
2.teil → group permissions
3.teil → other permissions

> d ... directory 
> l ... symbolic link 
> - ... file
> r ... read 
> w ... write 
> x ... execute 
> - ... keine rechte (leer)
> 4 ... read 
> 2 ... write 
> 1 ... execute
